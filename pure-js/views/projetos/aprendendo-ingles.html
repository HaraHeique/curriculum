Aprendendo Inglês

About Project

This is a really simple project made in Angular and with the main purpose of learning how to use the basics concepts.

Basically is a game where the user translates a couple of words in english to portuguese (Brazil). I made this simple application when I was learning Angular from a course on Udemy.
